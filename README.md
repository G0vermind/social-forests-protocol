# Social Forest 🌳
**A Decentralized RWA & DeFi Protocol (Stellar 37º - Track B)**

---

## 🇧🇷 Português: Escopo do Projeto
Protocolo de Finanças Regenerativas (ReFi) focado na convergência entre pagamentos tradicionais (Stripe) e lógica de ativos reais (Soroban). Atuamos na Trilha B para construir a base da economia de ativos biológicos na rede Stellar, integrando dados industriais da serraria **Sómogno** diretamente na blockchain.

### Roteiro de Sprints
* **Sprint 1:** Integração Stripe Checkout e arquitetura de cofres RWA.
* **Sprint 2:** Protocolo x402 para auditorias biométricas via IA.
* **Sprint 3:** Contratos Soroban (Rust) para Pools de Liquidez (AMM).

---

## 🇺🇸 English: Project Scope
Regenerative Finance (ReFi) protocol focused on the convergence of traditional payments (Stripe) and real-world asset logic (Soroban). We operate in Track B to build the foundation for the biological asset economy on the Stellar network, integrating industrial data from the **Sómogno** sawmill directly onto the blockchain.

### Sprint Roadmap
* **Sprint 1:** Stripe Checkout integration & RWA Vault architecture.
* **Sprint 2:** x402 Protocol for AI-driven biometric audits.
* **Sprint 3:** Soroban Rust contracts for Liquidity Pools (AMM).

---

## 🛠️ Tech Stack
* **Smart Contracts:** Soroban (Rust).
* **Payment Layer:** Stripe (MPP), x402 Protocol.
* **Frontend:** Next.js (App Router), Tailwind CSS.
* **Data Layer:** AI Engine (Biometric Processing) + Supabase.
