# FLORESTAS SOCIAL: Estratégia de Identidade Híbrida

O protocolo suporta dois caminhos de entrada:
1. **Login Social (Google/Email):** Usa Abstração de Conta para gerar chaves automáticas. O Florestas Social patrocina as taxas (Fee Bumps), criando uma experiência "Gasless" para o usuário Web2.
2. **Login Nativo (Multi-wallet):** Suporte total a Freighter e Albedo para usuários que exigem auto-custódia total.
