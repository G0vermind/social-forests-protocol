# 🌲 Guia de Contribuição - Social Forest

Bem-vindos à equipe de desenvolvimento! Estamos no **Antgravity** e nosso foco é excelência técnica na rede Stellar.

### 🛠 Nossa Stack
* **Smart Contracts:** Soroban (Rust)
* **Ativo Principal:** Mogno Africano (RWA)
* **Frontend:** Next.js (pasta /app)

### 🚀 Como colaborar?
1. **Branches:** Não suba código direto na `main`. Crie uma branch (ex: `feat/nome-da-tarefa`).
2. **Código:** Antes de enviar um contrato, garanta que ele compila.
3. **Documentação:** Comente funções complexas para facilitar a auditoria.

Dúvidas técnicas? Falem com o Gustavo no grupo ou abram uma "Issue" aqui no GitHub.
